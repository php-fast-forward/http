Coverage and Reports
====================

Live reports for the default branch are published here:

- `Coverage report <https://php-fast-forward.github.io/http/coverage/index.html>`_
- `Testdox report <https://php-fast-forward.github.io/http/coverage/testdox.html>`_

How To Use These Reports
------------------------

- Use the coverage report when you want to inspect which classes or paths are exercised by tests.
- Use the Testdox report when you want a behavior-oriented overview of the test suite in plain
  language.
